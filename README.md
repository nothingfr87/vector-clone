# Vector Clone

A clone of the vector in c++

---

A resizable array made in c using the dynamic memory allocation methods

---

## How to run

```bash
# Clone the repository 
git clone https://github.com/nothingfr87/vector-clone.git
cd vector-clone

# Build the source file
make build

# Run the binary file
./main
```
